# AskVuta

A modern, coastal-themed chatbot website for Vung Tau City, Vietnam's beautiful coastal tourism destination.

## Features

### Home Page
- **Hero Section**: Ocean-themed banner with gradient background and wave patterns
- **About Section**: Four key features of the chatbot (Tourism, Food, Culture, News)
- **News Carousel**: Scrollable news cards with different categories
- **Featured Places**: Popular tourist attractions in Vung Tau
- **Project Info**: Information about the founder and project goals
- **Footer**: Navigation links and social media contacts

### Chatbot Page
- **Interactive Chat Interface**: Modern ChatGPT-style UI
- **Quick Suggestions**: Preset questions for common queries
- **Smart Responses**: AI responds to questions about:
  - Restaurants and food
  - Tourist destinations
  - Local specialties
  - Hotels
  - Christ the King statue
  - Weather information
- **Typing Indicator**: Shows when bot is "thinking"
- **Responsive Design**: Works on desktop, tablet, and mobile

## Design Theme

- **Colors**: Ocean blue, turquoise, sand beige, white, sunset orange
- **Typography**: Poppins for headings, Inter for body text
- **Style**: Coastal, modern, friendly with glassmorphism effects
- **Animations**: Smooth transitions and hover effects using Motion (Framer Motion)

## Tech Stack

- React 18
- React Router for navigation
- Motion (Framer Motion) for animations
- Lucide React for icons
- React Slick for carousels
- Tailwind CSS v4
