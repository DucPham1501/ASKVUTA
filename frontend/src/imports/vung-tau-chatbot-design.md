Design a modern website UI for a Vietnamese chatbot about Vung Tau City (a coastal tourism city in Vietnam).

The website has 2 main pages:

1. Home Page
2. Chatbot Page

STYLE & THEME
The visual theme should reflect a coastal city atmosphere like Vung Tau:
- Ocean-inspired design
- Colors: ocean blue, turquoise, sand beige, white, and sunset orange accents
- Clean, modern, and friendly style
- Rounded UI elements
- Soft shadows and glassmorphism cards
- Smooth animations and transitions

TYPOGRAPHY
Use modern readable fonts suitable for Vietnamese:
- Heading: Poppins or Montserrat
- Body text: Inter or Roboto

BACKGROUND
Use subtle ocean themed backgrounds such as:
- soft wave patterns
- gradient sky-to-sea background
- light beach textures
- coastal illustrations

PLUGINS / ASSETS SUGGESTIONS
Use icon and illustration styles similar to:
- Heroicons
- Phosphor icons
- Simple coastal illustrations
- Travel themed icons

Grid card layout with scrollable content

- Quán ăn ngon

Each card includes:
- image

5. FOUNDER / PROJECT INFO
Simple section showing:

6. FOOTER
Standard website footer with:
